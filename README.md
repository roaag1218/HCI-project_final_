# HCI-project_final_
redesigning Egyptian National railway app

## Screens
- Login
- Signup for Egyptian users
- Sign up for foreigners
- Ticket store
- Search results
- Refund page

## Changes from Previous Version
- Redesigned Login screen adding sign in with google
- Allowing non-Egyptian users to use the app using passport number, issuing country and expiry date to authenticate the data
- Optimizing sign-up screen
- Clarifying verification options for ease of use 
- Optimized spacing for mobile screens
- Optimizing search results display: adding sort functionality and many options for varying tickets choices , clarifying each result display 
- Adding refunding feature so users can cancel their trips before a specified notice period
